# TinyIoT - very light oneM2M server

This is a very light oneM2M server using C

## OS

Linux Ubuntu 22.04

## Quick start

1. chmod 755 server
2. Run `./server` or `./server [port]` (port = 3000 by default).
3. If there is no executable file, Run `make`

## make Denpendency

1. Berkeley DB
2. cJSON
